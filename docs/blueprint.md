# **App Name**: Cloud Edit Pro

## Core Features:

- File Conversion: Convert files between various formats (e.g., image, document, audio, video).
- File Editing: Edit files with a range of tools like crop, resize, filters, trim, etc.
- File Compression: Reduce file sizes using various compression algorithms.
- File Management: Organize files in the cloud with folder structures and search functionality.
- Real-time Collaboration: Allow multiple users to collaborate on files simultaneously.
- Smart Suggestions: Provide AI-driven tool recommendations based on file type and desired modifications. LLM is used as a tool to decide whether to include format suggestions.
- Landing Page: Landing page with hero zone, format support, tool categories, live demo, trust badges, pricing, social proof, and FAQ.
- Sign Up/Login: Secure authentication with social auth, email/password, and workspace creation.
- Dashboard: Central command hub with real-time sync, file grid, smart suggestions, and activity log.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a professional and reliable feel.
- Background color: Light gray (#F0F2F5), a desaturated near-white that allows content to pop.
- Accent color: Purple (#7E57C2), an analogous hue for highlighting interactive elements.
- Body and headline font: 'Inter', a sans-serif font providing a modern and neutral look.
- Use a set of clear, modern icons to represent different file formats and actions.
- A clean, intuitive layout with a focus on usability and quick access to key features.
- Subtle animations for file uploads, downloads, and processing to enhance user experience.
- Primary brand gradient: Modern Blue → Purple (#2563EB → #7C3AED). Clean, professional, trustworthy. Best for dashboard + primary buttons.
- Accent gradient: Deep Purple → Pink (#7C3AED → #EC4899). Tech power, cloud intelligence, premium SaaS vibe, modern 2026 look.
- Micro interactions: 150–250ms; Page transitions: 300–400ms; Processing animations: continuous loop; Easing: Smooth ease-out for entry, Ease-in-out for UI movement, Linear for progress bars; Feel: “Precision & intelligence” — not bounce-heavy.
- Hero Section Animation: Gradient animated background (slow shifting), Floating file icons (PDF, JPG, DOC), Drag & drop glow effect. Logic: Files gently float, On hover → slight lift + shadow, Drag file → drop zone pulses. Purpose: Immediately show “smart file platform”.
- Upload Animation: States: Idle → soft dashed border, Dragging → glowing border + scale 1.02, Uploading → circular progress animation, Completed → checkmark morph animation. Progress ring fills smoothly, Percentage increments visually, Success icon animates in with fade + scale.
- Real-Time Processing Animation: Animated progress bar with moving gradient, “Optimizing…” dynamic text dots. Example: File Validated, Converting Format, Compressing, Securing, Completed - Each step fades in sequentially.
- Dashboard Animations: File Card Hover: Slight lift, Shadow increase, Action buttons fade in; Grid Loading: Stagger animation (files appear one by one); Sidebar: Smooth slide expand/collapse.
- Button Animation System: Primary Button: Gradient background shift on hover, Slight scale (1.03), Soft glow pulse. Click: Micro press effect (scale 0.97), Ripple or subtle wave.
- Conversion Screen Animation: Before Conversion: Input preview slides in from left, Settings fade up; During Conversion: Animated transformation preview.  PDF icon morphs into Word icon, Image resolution numbers change dynamically, File shrinks visually when compressing.
- Image Tool Animations: Resize: Numbers animate smoothly instead of jumping; Compression Slider: Real-time preview updates, Size counter counts down dynamically; AI Upscale: Before/After swipe slider animation.
- PDF Editor Animations: Page reorder drag smooth transition, Highlight tool glow, Text edit fade effect, Page add/remove slide animation.
- Batch Processing Animation: Queue System: Each file card shows: Pending (grey pulse), Processing (animated border), Completed (green check), Failed (shake + red); Queue moves automatically upward when completed.
- Notifications: Success: Slide from top-right → fade out; Error: Shake + red border highlight; Background Process Done: Bell icon pulse.
- Page Transitions: Between pages: Fade + slight upward motion; No harsh sliding; Keep minimal & clean.
- Loading States: Skeleton loaders; Shimmer effect; File outline pulse animation.
- AI Tool Animation: Animated scanning line (OCR); Typing effect for summary generation; Neural network pulse background (subtle).
- Gradient Animation System: Slowly shift direction; Move every 8–12 seconds; Very subtle (premium feel).
- Premium Touch Animations: Storage usage ring animation; Subscription upgrade glow effect; Crown icon shine (for Pro users); Secure lock icon pulse.
- Security Visual Effects: Encryption animation when uploading; Lock symbol appearing after processing; Shield ripple effect.
- Final Animation Strategy: Feel like: A Smart Cloud Engine Working In Real-Time; Not like: A design tool, A gaming app, A social media app.
